My Event Radar Web application

Built in a team of four. My event Radar is a web application built to simplify looking for local events in London that matchs the user interets. To Achieve that we built multiple features thinking on an easier user experience. One of our main goal was that it was not overwhelming for the user to find an event according to what they wanted. For that reason we created a search bar using PG search Ruby Gem which allow us to filter events by specifc category and name and we also integrated a Ruby date code which allow them to search by date. I worked on building an events carousel using Bootstrap for the carousel and cards and Ruby code for the logic of the cards appearing in groups of 3, this carousel would allow them to just see 3 events at the time and keep seeing more if those are not of their interest. Each card also contains a link to the specific event details. In addition, I also worked on building a chat for each event, the chat was built using stimulus js framework.
